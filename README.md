# Telegramapp

Telegramapp es una aplicacion de backend desarrollada con Node, Express y MongoDB que simula el comportamiento de Telegram

## Precondiciones
Instalación de Node https://nodejs.org/es/download/

## Instalación
npm i

## Iniciar la aplicación
npm start
